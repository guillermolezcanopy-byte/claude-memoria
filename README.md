# Fuego Cruzado 🔥

Juego de desafíos sexuales para parejas (+18). Mazo de **50 cartas** repartidas en
cuatro niveles de intensidad que podés combinar como quieras.

## Cómo jugar

1. Abrí `index.html` en cualquier navegador (celular, tablet o compu). No necesita
   internet ni instalación.
2. Elegí uno o varios niveles de picante.
3. Tocá **Empezar a jugar** y luego tocá la carta para revelar el desafío.
4. Cumplan, salten o reinicien el mazo cuando quieran.

## Niveles

| Nivel | Picante | Vibe |
|-------|---------|------|
| 😌 Suave | 🔥 | Juego previo, romántico, ropa puesta |
| 😏 Caliente | 🔥🔥 | Sube la temperatura de a poco |
| 🌶️ Picante | 🔥🔥🔥 | Sin vueltas, directo al fuego |
| 🔥 Infierno | 🔥🔥🔥🔥 | Lo más intenso del mazo |

## Estructura

- `index.html` — interfaz y estilos
- `cards.js` — el mazo de cartas (editá o agregá las tuyas acá)
- `game.js` — lógica del juego (barajado, niveles, navegación)

## Reglas de oro

Todo se juega entre adultos, con consentimiento y buena onda. Cualquiera puede
decir "paso" en cualquier momento, sin explicaciones. 💋
